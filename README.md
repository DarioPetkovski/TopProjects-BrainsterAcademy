# Arkitektur App

This app is one of the mentorships on the Brainster FrontEnd Academy. It's used as a static web site to see the static information coming from the api. For this app is used NextJs, Css, TypeScript, Bootstrap.

## Installation

### Clone this Repository

git clone https://github.com/DarioPetkovski/TopProjects-BrainsterAcademy/tree/Arkitektur-App

### Install dependencies

npm install

### Run Local Server

npm run server

### Run App

npm run start
